# wifi室内导航

## 基本设计

### 需求

1. 通过手机到不同Wifi的强度，计算手机的当前位置
2. 通过选择目标地址，画出路线
3. 当行动时，给出当前位置

### 输入

1. Wifi AP列表，需要每个AP的MAC地址
2. 需要提前收集每个AP在每个位置的Wifi信号强度
3. 标明障碍物的地图

### 单元模块

1. 图形化界面显示地图：能够进行位置的标记及显示，能够进行路线的显示
2. 获取位置：根据当前Wifi信号强度找到对应位置
3. 路线规划：根据初始位置和目的位置以及障碍物地图进行路线规划

### 系统功能

1. 需求1通过组合单元模块1、2完成
2. 需求2通过组合单元模块1、3完成
3. 需求3通过组合单元模块1、2完成
